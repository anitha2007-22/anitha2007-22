### 🎓 Smart Attendance Monitoring & Analytics System (SAMS)
> QR-based automated attendance system for colleges with real-time analytics

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Visit%20Site-2551eb?style=for-the-badge&logo=vercel)](https://smart-attendance-dusky.vercel.app/shared/login.html)
[![GitHub](https://img.shields.io/badge/GitHub-View%20Code-181717?style=for-the-badge&logo=github)](https://github.com/anitha2007-22/smart-attendance-system)

**Key Features:**
- 🔐 Rotating QR codes (expire every 20 seconds) for fraud-proof attendance

### 💳 PayWallet — FinTech Digital Wallet
> A full-stack digital wallet application with real-time notifications and fraud detection

[![Live Demo](https://img.shields.io/badge/Live%20Demo-Visit%20Site-16a34a?style=for-the-badge&logo=vercel)](https://paywallet-z99d.vercel.app)
[![GitHub](https://img.shields.io/badge/GitHub-View%20Code-181717?style=for-the-badge&logo=github)](https://github.com/anitha2007-22/paywallet)

**Key Features:**
- 🔐 User Authentication with JWT (Register & Login)
- 💳 Digital Wallet — Add Money & Withdraw
- 💸 Send & Receive Money instantly
- 📋 Transaction History with Search & Filter
- 📊 Analytics Dashboard with Charts
- 🔔 Real-time Notifications via Socket.IO
- 🚨 Fraud Detection System
- 🛡️ Admin Dashboard
- 🌙 Dark / Light Mode toggle
- 📱 Fully Mobile Responsive
